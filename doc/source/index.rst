Description du projet
---------------------

OC Lettings est une start-up dans le secteur de la location de biens immobiliers aux États-Unis.
Cette application permet de :

- Gérer un catalogue de biens immobiliers (lettings)
- Gérer les profils utilisateurs
- Administrer le contenu via l'interface Django Admin
